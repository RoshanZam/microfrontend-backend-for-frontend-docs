# Microfrontend Backend for Frontend Documentation

This repository contains documentation for the proposed architecture of **Micro Frontends combined with Backend for Frontend (BFF)**.

## Overview

Our product is transitioning from a monolithic backend architecture to a more modular and scalable approach using Micro Frontends and BFF. This documentation outlines the benefits, challenges, and implementation details of this architecture.

- [Architecture Overview](docs/architecture.md): Provides a detailed explanation of the Micro Frontends and BFF architecture, including its advantages and how it addresses common system issues.
